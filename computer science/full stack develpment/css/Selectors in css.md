In [[css]] There are many types of selector.

- group selector `h1, p, ...`
- id selector `#`
- class selector `.`
- attribute selector `[]`
- n-th child selector
- n-th type selector
- *  selector 
- Combination selectors
	- Decendant selectors(using space)
	- Chile selector (>)
	- General sibling selectors (~)
	- Adjacent sibiling selectors (+)
- [[pseudo-classes]] `::`
	- hover
	- active
	- focus
